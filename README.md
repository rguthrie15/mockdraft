# 2026 NFL Mock Draft Simulator

Full 7-round NFL Mock Draft simulator with:
- All 257 picks from the correct 2026 NFL draft order
- 257+ real 2026 prospects with scouting reports
- AI auto-drafting based on team needs
- Multi-team control
- Trade center (any round, any pick)
- Sim button to fast-forward AI picks
- Draft results view

## Run locally

```bash
npm install
npm start
```

## Deploy

Deployed via Vercel. Push to GitHub and Vercel auto-deploys.
